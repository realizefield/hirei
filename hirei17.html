📌 変域とは

変数がとることのできる値の範囲を 変域 といいます。
不等号を使って表します。

例）x が 2 以上 5 未満　→　2 ≦ x < 5
例）x が −3 より大きく 4 以下　→　−3 < x ≦ 4

言葉 記号 境界を含む？
以上≦✅ 含む
以下≦✅ 含む
より大きい<❌ 含まない
未満<❌ 含まない
📖 例題 1
y は x に反比例し，x = −3 のとき y = −12 である。
（1）y を x の式で表せ。
（2）x = −1 のときの y の値を求めよ。
（3）y = −2 となる x の値を求めよ。
（4）x の変域が −3 ≦ x ≦ −1 のとき，y の変域を不等号を使って表せ。

STEP 1　y を x の式で求める

xy = a に x = −3，y = −12 を代入：
a = (−3) × (−12) = 36
よって y = 36x

STEP 2　x = −1 のとき

y = 36 ÷ (−1) = −36

STEP 3　y = −2 のとき

xy = 36 に y = −2 を代入：
x × (−2) = 36　→　x = 36 ÷ (−2) = −18

STEP 4　変域を求める

y = 36x に x の両端を代入：
x = −3 → y = 36÷(−3) = −12（変域の端）
x = −1 → y = 36÷(−1) = −36（変域の端）

x が増えると y は 減る（反比例はy=a/xでa>0なら単調減少）
x: −3 ≦ x ≦ −1　→　y: −36 ≦ y ≦ −12

📖 例題 2　〈入試問題〉
y は x に反比例し，x = 2 のとき y = 4 である。
x の変域が −2 ≦ x ≦ −1 のとき，y の変域を不等号を使って表せ。

STEP 1　式を求める

a = 2 × 4 = 8　　よって y = 8x

STEP 2　x の両端を代入

x = −2 → y = 8÷(−2) = −4
x = −1 → y = 8÷(−1) = −8

STEP 3　変域を不等号で表す

−2 ≦ x ≦ −1 の範囲では，x が増えると y は減る。
x = −2 のとき y = −4（大きい方）
x = −1 のとき y = −8（小さい方）
よって　−8 ≦ y ≦ −4
