📖 例題
右の図のような長方形 ABCD の辺 BC 上を点 P が B を出発して C まで進む。
点 P が B を出発してから x cm 進んだときの三角形 ABP の面積を y cm² として，次の各問いに答えよ。
AB = 6 cm，BC = 12 cm

（1）y を x の式で表せ。
（2）x，y の変域を，それぞれ不等号を使って表せ。
（3）x と y の関係をグラフに表せ。
（4）三角形 ABP の面積が 25 cm² になるのは BP が何 cm のとき？

STEP 1　y を x の式で表す

三角形 ABP の面積 = 底辺 × 高さ ÷ 2
底辺 = BP = x cm，高さ = AB = 6 cm
y = x × 6 ÷ 2 = 3x

STEP 2　x と y の変域

P は B を出発して C まで進む。
BP = x は 0 より大きく 12 以下 → 0 < x ≦ 12
y = 3x に代入：
x = 0 のとき y = 0（境界除く）
x = 12 のとき y = 36
→ 0 < y ≦ 36

STEP 3　グラフの特徴

y = 3x は原点を通る右上がりの直線。
ただし変域 0 < x ≦ 12 の部分のみ。
x = 0 は含まない（白丸○）
x = 12 は含む（黒丸●）
◯ = 端点を含まない　● = 端点を含む

STEP 4　面積が 25 cm² のとき

y = 3x に y = 25 を代入：
25 = 3x　→　x = 25 ÷ 3 = 253 cm
